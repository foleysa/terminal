Why IT Costs Keep RisingANALYSIS
Perpetual → Subscription: The Industry-Wide Shift
Software vendors have systematically moved from perpetual licenses (buy once) to annual subscriptions (pay forever). Microsoft 365, Adobe Creative Cloud, Autodesk — all subscription now. What was a $500 one-time purchase is now $300/year forever. Over a 5-year lifecycle, the subscription costs 3x the perpetual license. And vendors raise subscription prices 5-10% annually because switching costs make customers sticky.
Cloud Spend — The New Unmanaged Category
Cloud (AWS, Azure, GCP) is the new "nobody manages it" spend category. Developers spin up resources, forget to shut them down, over-provision instances, and nobody optimizes. Gartner estimates 25-40% of cloud spend is wasted. A $500K annual cloud bill likely has $125K-200K in optimization opportunities — idle instances, oversized VMs, unused storage, and missing reserved instance commitments.
Telecom — The Declining CategoryFCC · BLS
Voice & WAN Costs Are Declining
Traditional telecom (voice lines, MPLS, PRI trunks) is declining 3-5% annually. VoIP/UCaaS (Teams, Zoom, RingCentral) replaces PBX hardware and PSTN lines. SD-WAN replaces expensive MPLS at 40-60% lower cost with comparable or better performance. If you're still paying for MPLS circuits and PRI trunks, you're overpaying for technology that the market has moved past.
Endpoint Refresh Cycles — Extending
PC refresh cycles have extended from 3-4 years to 4-5 years as hardware quality improves. But Windows 10 end-of-support (October 2025) forces a hardware refresh for PCs that can't run Windows 11 (TPM 2.0 requirement). This is a one-time cost spike — budget for it but don't let Dell/HP/Lenovo use the urgency to overcharge. Get competitive bids from all three.

⚠ Broadcom / VMware — Price Shock
Broadcom acquired VMware in November 2023 and immediately restructured licensing: eliminated perpetual licenses, moved to subscription-only bundles, and raised prices 2-10x for many customers. If you run VMware vSphere, your renewal will be dramatically higher. Evaluate alternatives NOW: Microsoft Hyper-V (free with Windows Server), Nutanix AHV, or Proxmox. The migration is painful but the alternative is accepting Broadcom's pricing monopoly on your entire virtualization infrastructure.
⚠ Oracle Java Licensing
Oracle changed Java SE licensing in 2023 to per-employee pricing (not per-installation). A 1,000-employee company using Java on even one server now owes Oracle $15/employee/month = $180,000/year. Many companies don't realize they owe this. Alternatives: OpenJDK, Amazon Corretto, Eclipse Temurin — all free, all production-ready. Migrate off Oracle Java before the audit letter arrives.

Software License & SaaS OptimizationFSA ANALYSIS
The SaaS Sprawl Problem
The average mid-market company (500-5,000 employees) has 130+ SaaS subscriptions. 25-30% are unused (licenses assigned to departed employees or never-adopted tools). 15-20% are redundant (three project management tools, two CRMs, multiple file-sharing platforms). And every one auto-renews annually. Nobody is managing this portfolio — IT buys some, departments buy some on credit cards, nobody reconciles. The result: $50K-500K/year in SaaS waste.
License Traps
MICROSOFT EA TRUE-UP
Microsoft Enterprise Agreements require annual true-ups — counting actual usage vs. contracted licenses. If you've added users or deployed products beyond your agreement, you owe back-pay at list price (no discount). True-up audits catch companies with 20-50% more usage than contracted. Count your deployments BEFORE the true-up deadline.
ORACLE AUDIT
Oracle's licensing model (processor-based, named user plus, with complex virtualization rules) is designed to create non-compliance. Oracle audits more aggressively than any other vendor. The typical audit finding: $500K-5M in unintentional non-compliance. Defense: maintain a Software Asset Management (SAM) tool, document your deployment architecture, and engage a third-party Oracle licensing advisor before the audit, not after.
AUTO-RENEWAL — EVERYWHERE
Every SaaS contract auto-renews 30-90 days before expiration. Miss the cancellation window and you're locked in for another year at whatever price the vendor set. Calendar EVERY renewal date. File cancellation notices 90 days out even for products you plan to keep — it forces a negotiation.
Optimization Playbook
STEP 1 — INVENTORY EVERYTHING
Pull 12 months of AP/credit card data. Search for every recurring software charge. Build a master list: vendor, product, annual cost, number of licenses, contract end date, auto-renewal terms. Most companies discover 30-50% more subscriptions than they knew about.
STEP 2 — MEASURE USAGE
For every product, check actual usage: active users vs. licensed users, login frequency, feature adoption. Most SaaS admin portals have usage dashboards. If 40% of your Salesforce licenses haven't logged in for 90 days, you're paying for 40% more seats than you need.
STEP 3 — ELIMINATE REDUNDANCY
Map products to functions: how many project management tools? How many video conferencing platforms? How many file-sharing solutions? Standardize on one per function and eliminate the rest. Common redundancies: Slack + Teams, Dropbox + OneDrive + Google Drive, Monday + Asana + Jira.
STEP 4 — NEGOTIATE RENEWALS
Armed with usage data and competitive alternatives, negotiate every renewal. Tactics: right-size to actual usage, request multi-year discount (15-25% typical), time negotiations to vendor fiscal year-end, and always have a credible alternative. "We're evaluating [competitor]" is the most powerful sentence in software negotiation.

Cloud Cost Optimization — AWS / Azure / GCPFSA ANALYSIS
25-40% of Cloud Spend Is Wasted
Gartner and Flexera both estimate 25-40% of enterprise cloud spend is wasted. The causes: idle instances running 24/7 that are only needed 8 hours/day, oversized VMs (production workloads on instances with 4x the CPU/RAM they need), unattached storage volumes, forgotten dev/test environments, and failure to use reserved instances or savings plans for predictable workloads.
Where The Waste Hides
IDLE INSTANCES — 20-30% OF COMPUTE
Dev/test environments running 24/7 when they're only used during business hours. Auto-shutdown schedules (AWS Instance Scheduler, Azure Auto-Shutdown) eliminate 60-70% of dev/test compute cost overnight and on weekends.
OVERSIZED VMs — 40%+ OF INSTANCES
The default is to overprovision. A workload that needs 2 vCPUs and 4GB RAM runs on a 4 vCPU / 16GB instance "just in case." AWS Compute Optimizer and Azure Advisor both recommend right-sizing based on actual utilization. Average savings: 30-50% per right-sized instance.
NO RESERVED INSTANCES — PAYING ON-DEMAND
On-demand pricing is 40-60% more expensive than 1-year reserved instances or savings plans. For any workload running consistently (24/7 production), reserved instances are the single biggest savings lever. A $100K annual on-demand spend becomes $40K-60K with a 1-year reservation commitment.
Optimization Actions
USE NATIVE TOOLS — FREE
AWS: Cost Explorer, Compute Optimizer, Trusted Advisor.
Azure: Cost Management + Advisor.
GCP: Cost Management + Recommender.
Every cloud provider gives you free tools to find waste. If you haven't looked at your cloud provider's cost recommendations dashboard, start there.
COMMIT TO RESERVATIONS
For any workload running 24/7 for the next 12 months: buy a 1-year reserved instance or savings plan. 40-60% savings vs on-demand. Start with production databases and application servers — these are the easiest to predict and the biggest on-demand costs.
IMPLEMENT TAGGING & CHARGEBACKS
Tag every cloud resource with owner, project, and environment (dev/test/prod). Implement monthly chargeback reporting so business units see their cloud costs. When the marketing team sees they're paying $3K/month for a dev environment nobody uses, they shut it down. Visibility drives accountability.
NEGOTIATE EDP / COMMIT
AWS Enterprise Discount Program (EDP), Azure MACC (Microsoft Azure Consumption Commitment), and GCP Committed Use Discounts all offer 10-20% additional savings on top of reserved instances for large spend commitments. If your annual cloud spend exceeds $250K, you qualify for committed-spend discounts.
